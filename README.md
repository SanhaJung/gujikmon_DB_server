# gujikmon_DB_server





## pip Install

* 가상환경

  ``` bash
  pip install -m venv gujikmon
  ```

- 장고

  - ```bash
    pip install django
    ```

- djongo

  - mongodb와 연결하기위한 라이브러리

  - ```bash
    pip install djongo
    ```

* 파이썬 HTTP 요청처리를 위한 모듈

  ``` bash
  pip install requests 
  ```

* xml을 dict로 바꾸는 모듈

  ``` bash
  pip install xmltodict
  ```
